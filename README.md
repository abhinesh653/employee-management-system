# Employee Management System

A backend REST API built using Spring Boot and MySQL for managing employee records.

## Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- postman 

## Features

- Add new employee
- View all employees
- Get employee by ID
- Update employee details
- Delete employee
- Exception handling for employee not found

## API Endpoints

POST /employees  
Create a new employee

GET /employees  
Get all employees

GET /employees/{id}  
Get employee by ID

PUT /employees/{id}  
Update employee

DELETE /employees/{id}  
Delete employee

## Project Structure

controller – Handles API requests  
service – Business logic  
repository – Database operations  
model – Entity classes  
exception – Custom exception handling
